# Esthetica

## About
Esthetica is a platform for artists to showcase their work and for art lovers to discover and appreciate art. It is a place where artists can share their work and get feedback from the art community. It is also a place where art lovers can discover new and upcoming artists, as well as their work.

## Author
[Shaniz Nyawira](https://github.com/Shaniz-Nyawira/')

## Technologies
      - HTML
      - CSS
      - JavaScript
      - Web API

## Setting Up
      $ git clone https://github.com/Shaniz-Nyawira/Esthetica.git
      $ cd Esthetica
      $ google-chrome index.html
